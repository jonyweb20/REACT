//Nodejs - программа платформа, основанная на движке V8, превращающая JS из узкоспециализированного npm - node package manager. Консольная утилита (CLI), которая позволяет управлять пакетами (модулями, библиотеками) в вашем проекте, устанавливается вместе с Node. Команды npm: npm init - инициализирует начальный проект npm i название модуля - устанавливает модуль в проект и позволяет установить все модули указанные package-lock.json ЭТО ОЗНАЧАЕТ. ЧТО ПАПКУ NODE_MODULES не нужно заливать на Gid(флешку) npm i nodemon модуля -D - установка пакета через зависимости для разработки (-D develop)
